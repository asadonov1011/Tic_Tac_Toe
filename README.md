# Tic-Tac-Toe Game 🎮

A simple, interactive Tic-Tac-Toe game built with web technologies. This is my first project hosted on GitHub!

## 🚀 Overview
This is a classic two-player Tic-Tac-Toe game. It features a clean interface and logic to determine the winner or a draw.

## 🛠 Technologies Used
- **HTML5**: For the game structure.
- **CSS3**: For styling the game board and animations.
- **JavaScript**: For the game logic and player turns.

## 🕹 How to Play
1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Take turns clicking on the squares to place your mark (X or O).
